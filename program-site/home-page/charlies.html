My initial design was difficult for me to code for my first coding project, so I had to simplify my design as you can see. I designed three different size sites: one for mobile sizes, one for tablet sizes, and one for desktop sizes. As I just learned to code, this project became very challenging for me, but I learned a lot along the way.
