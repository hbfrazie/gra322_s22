SYMBOL MOTION ANIMATION

For this project we were tasked with animating the logo we created for our Letter Number Combination project. For this project I was assigned the letters C, L, and the number 1. I ended up designing a sharp/geometric logo, which I felt fit the gaming industry well. At first I found it hard to adapt to the animating process in After Effects, but quickly found myself having fun with it, and learned so much from delving into animation.
